Update Flow

Routine update

Use this only on devices already provisioned with Spectacle. It flashes the app and SPIFFS images, but it does not repair the partition table.

Use Chrome or Edge on desktop with a data-capable USB cable. Before connecting, press both side buttons to enter flashing mode. The StickS3 is ready when the green LED starts blinking. If the device is fresh, fully erased, or boots with SPIFFS FAILED, use factory reflash instead.

    Use this for first install, after a full erase, or when the device is corrupted, the flash layout needs repair, or the boot screen shows SPIFFS FAILED.

    This path erases the entire chip and then restores the bootloader, boot metadata, partition table, application, and SPIFFS images. Enter flashing mode the same way: press both side buttons until the green LED blinks.
